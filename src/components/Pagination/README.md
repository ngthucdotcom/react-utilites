# Pagination

## Usage

```tsx
<Pagination
	page={<current page>}
	rowsPerPageOptions={[<accept_list_number_option>, {label: 'Tất cả', value: -1}]}
	totalRecords={<total_records_or_list_length>}
	rowsPerPage={<row_per_page >}
	onPageChange={<function_handle_page_change>}
	onRowsPerPageChange={<function_handle_rows_per_page_change>}
/>
```
